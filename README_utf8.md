# Lab 9 Assignment - RSCU Project

## Overview
This project analyzes Relative Synonymous Codon Usage (RSCU) for biological sequences, specifically focusing on replicase and spike protein genes.

## Project Description

This assignment involves calculating and analyzing codon usage bias in viral proteins using the RSCU (Relative Synonymous Codon Usage) metric.

### Key Concepts
- **RSCU**: A measure of codon usage bias that accounts for the number of codons available for each amino acid
- **Codon Usage**: The frequency at which different codons are used in a given organism or gene

## Usage

Use the input files in **data/** and the reference material in **docs/reference/** to analyze codon usage according to the assignment specifications.
